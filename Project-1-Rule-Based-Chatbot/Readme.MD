# Project 1 - Rule-Based AI Chatbot 

## Overview

This project was developed as part of the DecodeLabs Artificial Intelligence Internship Program.

The chatbot is a rule-based conversational system that responds to predefined user inputs using conditional statements and decision-making logic. It demonstrates the fundamental concepts behind artificial intelligence systems before introducing machine learning techniques.

## Objectives

* Understand rule-based AI systems
* Practice control flow and decision-making
* Implement continuous user interaction
* Build a chatbot using Python

## Features

* Handles greetings
* Responds to user questions
* Displays a help menu
* Handles exit commands
* Uses continuous conversation loops
* Case-insensitive input processing
* Tracks user interactions

## Technologies Used

* Python 3

## Concepts Applied

* Variables
* Functions
* Loops
* Conditional Statements
* String Manipulation
* Rule-Based Artificial Intelligence

## How to Run

1. Open a terminal in the project folder.
2. Run:

python chatbot.py

## Sample Interaction

You: hello

Bot: Hello! Nice to meet you.

You: what is your goal

Bot: My goal is to assist users and demonstrate rule-based artificial intelligence.

You: help

Bot: Here are the commands I understand...

You: bye

Bot: Goodbye!
